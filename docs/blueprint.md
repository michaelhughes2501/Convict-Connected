# **App Name**: ConvictConnect

## Core Features:

- Profile Creation: Users can create profiles detailing their background (including 'crimes'), interests, and what they're looking for.
- Inmate Search: Users can search for potential matches based on various criteria such as location, shared interests, or crime type.
- Kite Messaging: Users can send messages (kites) to each other, initiating conversations and building connections.
- Match Recommendation: The app uses a matching algorithm to suggest potential matches based on user profiles and preferences.
- Virtual Visits: Users can schedule and conduct virtual visits within the app. AI will determine, based on their history, whether a pair is ready for a virtual visit, acting as a kind of trust tool.
- Release Planning Resources: Connects users with resources to assist in their transition upon release, such as job boards or halfway houses.
- Community Forum: A space for users to discuss shared experiences, seek advice, and build a sense of community. Content is heavily moderated.

## Style Guidelines:

- Primary color: Desaturated violet (#A895BC) for a calming yet distinctive feel.
- Background color: Very light desaturated violet (#F4F2F7).
- Accent color: Dusty rose (#BC95A8), a closely related color with warmth to draw the eye to key elements.
- Body and headline font: 'Inter' sans-serif font, for a clean and modern look.
- Use simple, minimalist icons. Focus on clear communication rather than embellishment.
- Clean and straightforward layout with a focus on user-friendliness. Prioritize easy navigation.
- Subtle transitions and animations to provide feedback without being distracting.
